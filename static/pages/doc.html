Background

The incidence of Lyme disease has been increasing in recent years. We started this project with the goal of examining that increase and analyzing how ecological factors may be contributing. Data was reviewed for the entire United States, but the primary focus was on New Jersey. This was because New Jersey has a higher prevalence of Lyme disease than many other states and it is local to our team. We identified the following questions as guides for research in determining trends for the incidence of Lyme disease:

  • Is Lyme disease more common in counties with higher deer populations?
  • Does deer hunting / population control reduce Lyme disease?
  • Is warm weather a predictor in the following year's Lyme disease rates?
  • Do oak mast years cause an increase in Lyme disease the following year?

Tasks

Find and extract source data
The first step was identifying data sources and extracting the data from them.
Data cleaning and normalization
From our raw data inputs we cleaned and normalized the data with the goal of setting up a reference database.
Database management
Normalized data was then loaded into a Postgres development database prior to being converted into a Heroku database.
Analysis and visualization
Visualizations were built concurrently by different team members, with some charts being worked on in teams. We aimed to use a variety of JavaScript libraries to build our visualizations.
Build dashboard and website
The dashboard function of the website was built around the interative visualizations before the website was finished with styling and written content.

Limitations

  • Data was not completely available for all counties and / or years for NJ.
  • The number of ticks in an area was not available, just if a tick presence was established.
  • Lyme disease cases were based on residency of the affected subject rather than where it was contracted.
  • Mast index not recorded by NJ, we had to use local news reporting to identify mast years, which may be incomplete.
  • Monthly weather data was too granular for our time frame to fully analyze.
  • Deer population data had to be extrapolated for certain years.
  • Deer harvest data was based on Deer Management Zones, some of which spanned multiple counties so assumptions had to be made in assigning a single county to each zone.
